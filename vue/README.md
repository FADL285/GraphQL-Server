# Vue App - GraphQL Client UI

Vue 3 application with Composition API for interacting with the GraphQL server.

## Features

- Vue 3 with Composition API
- TypeScript support
- Simple, clean UI
- Test data for development

## Project Structure

```
vue/
├── src/
│   ├── components/    # Vue components
│   ├── App.vue        # Main app component
│   ├── main.ts        # Entry point
│   └── assets/        # Static assets
├── public/            # Public assets
└── vite.config.ts     # Vite configuration
```

## Installation

From the root directory:

```bash
npm install
```

Or from this directory:

```bash
npm install
```

## Running the Vue App

### Development Mode

```bash
# From root
npm run dev:vue

# Or from this directory
npm run dev
```

The app will be available at `http://localhost:5173` (or check the terminal output).

### Build for Production

```bash
# From root
npm run build:vue

# Or from this directory
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Development

- TypeScript for type safety
- Vite for fast development
